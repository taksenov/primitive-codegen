# primitive-codegen

## Description

Codegeneration for ReactJS Application and/or standalone components, containers,
states etc

Ru: Кодогенерация компонентов для моих ReactJS проектов. На основе паттерна
"Стратегия"

### How use

Используется `ts-node`

Команды запуска:

```
npm i
npx ts-node ./src/main.ts --help
```
